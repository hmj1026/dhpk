# Workflow Checklists

本檔改為薄索引；長版 checklist 已拆到 workflow-specific references，避免 `lightweight` 情境也把 feature/bugfix 的細節一起載入。

## Read This Only When

- 你需要快速找 checklist 入口，但還沒確定要讀哪一份
- 你正在維護舊連結

## Checklist Map

- `workflow-feature-delivery.md`
  - 新功能、行為變更、跨模組調整
- `workflow-bugfix.md`
  - 錯誤調查與修復
- `workflow-lightweight.md`
  - 不改行為的小修
- `handoff-and-verification.md`
  - 回報 gate 狀態與下一步
- `script-operations.md`
  - 需要具體命令時才讀
- `projects-index.md`
  - 需要 project-specific 範例或 shortcut 時再讀
